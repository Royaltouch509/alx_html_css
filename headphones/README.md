# Headphones Website

A responsive, modern website for a premium headphones brand built with pure HTML5 and CSS3 ‚Äî no JavaScript, no external frameworks.

##  Features

- **Fully Responsive Design**: Automatically switches to mobile layout at 480px screen width or less
- **Modern UI/UX**: Clean design with intuitive navigation and engaging visual elements
- **Pure CSS Implementation**: No JavaScript or external libraries used
- **Accessibility Ready**: Semantic HTML structure with proper heading hierarchy
- **Performance Optimized**: Lightweight and fast-loading

##  Design Specifications

### Typography
- **Primary Font**: Source Sans Pro (fallback to system sans-serif)
- **Decorative Font**: Spin Cycle OT (for special elements)
- **Font weights**: Regular, Bold, Light for visual hierarchy

### Color Palette
- **Primary Background**: `#0D0D1A` (dark navy blue)
- **Accent Color**: `#FF6565` (vibrant coral/pink)
- **Text Colors**: White (`#FFFFFF`) for dark backgrounds, Dark text (`#333333`) for light backgrounds
- **Interactive Elements**: 
  - Links hover/active: `#FF6565`
  - Buttons hover/active: `opacity: 0.9`

### Layout Constraints
- **Maximum Content Width**: 1000px (centered on page)
- **Mobile Breakpoint**: 480px and below
- **Responsive Grid**: Flexbox and CSS Grid for modern layouts

##  Mobile Responsiveness

The website automatically adapts to different screen sizes:
- **Desktop (1001px+)**: Full-width design with 4-column grids
- **Tablet (481px - 1000px)**: Adjusted layouts with 2-3 columns
- **Mobile (480px and below)**: Single column, simplified navigation, larger touch targets

##  Browser Compatibility

- **Modern Browsers**: Chrome, Firefox, Safari, Edge (latest versions)
- **Mobile Browsers**: iOS Safari, Android Chrome
- **CSS Support**: Flexbox, Grid, Custom Properties, Media Queries
